# ITOP - Intelligent Task Orchestration Platform

AI-powered task orchestration system with multi-agent architecture.

## Architecture
- 3 Java Spring Boot services
- 2 Node.js services  
- 2 Python FastAPI services
- React TypeScript frontend

## Tech Stack
- Backend: Java, Node.js, Python
- Frontend: React + TypeScript
- AI/ML: Groq/Ollama, LangChain, ChromaDB
- Infrastructure: Docker, PostgreSQL, Redis, Kafka
